<h1 align="center"> Senac Login Form </h1>

The main goal of this project is to provide a foundation for creating a functional login form in a web application, from structuring HTML elements to visual styling and data validation with JavaScript. While it does not authenticate with the actual Senac website, it offers a learning opportunity for developers who wish to create login forms in their own projects.

## Screenshots

![Screenshot](/src/imgs/project.png)

## How to Use

To use this project, you need to follow these steps:

1. Download or clone this repository to your computer.
2. Open the `index.html` file in a web browser of your choice.
3. Enter your username and password in the respective fields.
4. Click the 'Enter' button to proceed with the process.

## How It Works

"This project is a simulation of a login form inspired by the Senac website, a Brazilian educational institution. Although this form does not have real authentication functionality with Senac, it was created to illustrate how a typical login form can be implemented in a web application."

## Resources Used

This project was developed using the following technologies:

- **HTML**: Responsible for the structure of the login form.
- **CSS**: Used for styling the elements and providing a pleasant appearance.
- **JavaScript**: Implements the data validation logic and form interactivity.
